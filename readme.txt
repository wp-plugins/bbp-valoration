Adds thumbsup, visits and replies count to topics on bbpress, display results on a widget.

== Description ==

Adds thumbs up vote for topics on BBpress forum, also counts the number of visits and replies for each. Show Topics on a Widget filtered by more voted, visited or more replied, also allow to show or not the numbers.

Major features in bbp-valoration include:

* Add a Thumbs-up to the topics for user to like.
* Allow only one like per user.
* Counts the visits.
* Counts replies.
* Use a Widget to display the topics, more voted, more visited and more replied.
* Display or not the number values
* Uses Ajax.

PD: This plugin works only with a BBpress forum installation.

== Installation ==

Upload the bbp-valoration plugin to your blog, Activate it.

1, 2, 3: You're done!
